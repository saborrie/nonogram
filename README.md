# Nonogram App

Contains

- A component library using Materal UI and React Storybook.
- A React app using `create-react-app`, `react-router` and `immutable`

How to run the app:

1.  ```
    cd shared-dependencies
    yarn
    yarn setup
    ```

2.  ```
    cd nonogram-components
    yarn
    yarn link react
    yarn build
    ```

3.  ```
    cd nonogram-web-app
    yarn
    yarn start
    ```
